CSCE 425/825 - Compiler Construction

Scanners and Parsers

Exercises

  1. Implement and test a flex based scanner that:
    • formats text in 80-column blocks, while removing multiple, leading, or trailing blanks;
    • removes all tags from an HTML document; or
    • introduces plausible spelling errors.

    If you have trouble with unresolved references to yywrap try using %option noyywrap at the top of your flex input file.
     

  2. Read the flex documentation and discuss the full syntax for regular expression. Describe the differences for JLex (if any).
     
  3. Find out about "start conditions" in flex.
     
  4. Solve Exercise 3.3 in Appel's book (page 85).
     
  5. Solve Exercise 3.12 in Appel's book (page 87).
     
  6. Trace the action of the example parser from class on the input x + x + x.

Recommended Reading

  • Appel, Chapter 2 (skip whatever you know from previous courses).
  • Appel, Sections 3.1, 3.3 (skip SLR), 3.4.
