fstore

Operation
Store float into local variable

fstore
index

Forms
fstore = 56 (0x38)

Stack
..., value ...

Description
The index is an unsigned byte that must be a valid index into the local variables of the current frame (§3.6). The value on the top of the operand stack must be of type float. It is popped from the operand stack, and the value of the local variable at index is set to value.

Notes
The fstore opcode can be used in conjunction with the wide instruction to access a local variable using a two-byte unsigned index.
