i2c

Operation
Convert

i2c
int to char

Forms
i2c = 146 (0x92)

Stack
..., value ..., result

Description
The value on the top of the operand stack must be of type int. It is popped from the operand stack, truncated to char, then zero-extended to an int result. The result is pushed onto the operand stack.

Notes
The i2c instruction performs a narrowing primitive conversion (§2.6.3). It may lose information about the overall magnitude of value. The result (which is always positive) may also not have the same sign as value.
