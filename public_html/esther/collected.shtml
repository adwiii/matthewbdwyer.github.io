Eclipse universal tool platform (SDK 3.2) -- http://eclipse.org
  • Make sure that the Eclipse Java compiler generates Java 5-compliant class files (the ASM and Bytecode outliner plugin does not support Java 6). To set this, open the Window->Preferences... menu item. Expand Java in the preference tree, and select Compiler. Make sure that the Compiler compliance level is set to 5.0.
