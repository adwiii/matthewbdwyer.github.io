§1.54 Class NoSuchMethodError

 
public class java.lang.NoSuchMethodError
    extends java.lang.IncompatibleClassChangeError (§1.48)

    public NoSuchMethodError();	§1.54.1
    public NoSuchMethodError(String s);	§1.54.2

The Java runtime throws an instance of this class if the application tries to call a specified method of a class (either static or instance),and that class no longer has a definition of that method.

Normally, this error is caught by the compiler; this error can only occur at runtime if the definition of a class has incompatibably changed.
