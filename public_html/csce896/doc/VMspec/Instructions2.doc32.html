dload

Operation
Load double from local variable

dload
index

Forms
dload = 24 (0x18)

Stack
... ..., value.word1, value.word2

Description
The index is an unsigned byte. Both index and index + 1 must be valid indices into the local variables of the current frame (§3.6). The local variables at index and index + 1 together must contain a double. The value of the local variables at index and index + 1 is pushed onto the operand stack.

Notes
The dload opcode can be used in conjunction with the wide instruction to access a local variable using a two-byte unsigned index.
