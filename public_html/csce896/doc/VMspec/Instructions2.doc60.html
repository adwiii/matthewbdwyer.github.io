fload

Operation
Load float from local variable

fload
index

Forms
fload = 23 (0x17)

Stack
... ..., value

Description
The index is an unsigned byte that must be a valid index into the local variables of the current frame (§3.6). The local variable at index must contain a float. The value of the local variable at index is pushed onto the operand stack.

Notes
The float opcode can be used in conjunction with the wide instruction to access a local variable using a two-byte unsigned index.
