Coding

  • Implement a flow analysis to calculate POSTDOM and use it to calculate control dependence information - this will require some additional graph traversal to compute. Recall that control dependence is defined such that n is CD on m if:
    1. there exists a path, p = [m ... n], such that for all v on p, except for m and n, n is in POSTDOM(v); and
    2. n is not in strict POSTDOM(m)
    i.e., there is a choice from m that forces n to execute and there is a choice that avoids execution of n entirely.
  • Write a series of small Java programs that illustrate the differences between Soot's Shimple (SSA) and Jimple intermediate representations.
  • Can you characterize when Jimple is equivalent ot Shimple in terms of source program structure?
